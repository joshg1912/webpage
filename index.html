Mondrian Painting Generator

Mondrian painting was coded using HTML Canvas's Rect() function. A total of three paintings were drawn, each within it's own function. Using Math.random gives creative control to the computer, generating a different painting with different colors each time the page is refreshed.

Javascript Sleep Calulator

Sleep calculator utilizes conditionals in javascript to display a user's sleep debt. If a user's actual sleep hours per week is equal to their ideal sleep hours per week, they are on track. If there is a deficienct or surplus, sleep calculator will let the user know how many hours need to be added or taken away.
